## Installation
- Clone This Repository
`git clone https://github.com/monzuralam/all-in-one-content-protection-shopify-app.git`
`cd all-in-one-content-protection-shopify-app`

- Composer Install
`composer install`

- NPM Install
`npm install`

- Start Laravel development server
`php artisan serve`

- Run npm development build
`npm run dev`

- Start ngrok with the specified domain
`ngrok http --domain=seal-great-wren.ngrok-free.app 8000`
